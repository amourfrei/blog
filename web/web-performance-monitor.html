    # 前端性能监控平台

    # 功能集合

    1. 真实用户体验 -- 实时采集真实用户体验数据，端到端多维数据分析展示
    2. 资源详情 -- 采集资源加载详情，快速统计、评测资源质量、监控CDN性能
    3. 脚本错误 -- 采集用户浏览器报错信息，收集业务中不可预测的错误
    4. 深入交互 -- 全量抓取ajax数据，深入用户交互，数据交互可视化
    5. 页面加载 -- 记录页面加载全过程，建立页面和资源的加载时序图
