    # 京东会员码小程序插件

    # 描述

    • 小程序会员码插件是将CRM带有会员身份的二维码、条形码赋能到合作伙伴的小程序中，方便合作伙伴小程序端快速接入。
    • 帮助合作伙伴生成并显示安全的会员码（ 二维码和条形码 ）。

    # 功能

    1. 会员码展示
      • 定时刷新
      • 二维码放大
      • 防截屏
      • 风控保障
      • 验证安全保障
    2. 京东、微信免密支付展示
    3. 微信免密支付开通
    4. 支付设置
    5. 会员码刷新

    # 接入

    1. 在小程序后台搜索「京东支付会员码」并申请使用，通过后即可使用

    # 使用方法

    # 引入插件

    //@version 根据要接入的版本填写
{
    "plugins": {
        "memberCode": {
            "version": "1.0.0",
            "provider": "wxb************"
        }
    }
}

    使用自定义组件：

    //使用wx:if控制离开组件时销毁
<memberCode wx:if="{{ pluginStatus }}"></memberCode>

    使用小程序插件内api：

    • 定义回调函数
    /**
**@params:
**@name: options 
**@type: Object
**/
function jdpayPluginCallback ( options ){
    //your code
}

//options格式：
{
    code: 0, //0：使用webview打开指定链接 1：开通微信免密 2：用户未登录 3：返回上一页
    type: 0, //0： 开通京东免密 2：支付设置 3：支付成功，仅当code为0 时使用
    url: '', //要打开的页面url
    miniPayParam: {} //开通微信免密所需数据
}

    初始化插件api:

    // pt_key和sid为业务登录标识，根据自己业务选择使用，有且仅有一个
// deviceId为设备号，必传
// openid为小程序内风控所需字段，必传
var params = {
    sid: '',
    deviceId: '',
    openId: ''
}
plugin.init( param, jdpayPluginCallback )

    小程序后台配置request合法域名

    1. 开通京东免密支付的域名： https://mapi.jdpay.com/
    2. 支付设置的域名： https://payxsign.jd.com/
